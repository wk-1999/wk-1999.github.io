【一个筐】第一次博客建站缘由

2020-04-03
一个筐 / 本站
字数统计: 534   |   阅读时长≈ 1 分钟

　　说起来，这个博客是之前做云盘 index 的副产品。

　　一切都因为自己想要搞一个安全好用的 office 账号，要有 onedrive 空间。年后我发现有人说 E5 有渠道可以注册了，于是就找了教程搞了一个 E5 账号，但是只有 3 个月期限，需要自己找方法调用 API 才可以不断续期。

　　然后就参考了游轶老哥的教程，买了域名，租了服务器，搭建了 Oneindex 做私人网盘。但是服务器加域名每年也得一百多块钱，只搞个 Oneindex 有点浪费，为了尽可能利用好租的服务器，就进一步部署了 WordPress，搭建起了个人博客。这方面我是小白，参考了网上很多教程，前后花了三四天才调教完成。

　　通过自建博客，我发现自己很喜欢这样的记录形式，一切都在自己掌控之下，不用担心微信公众号、QQ 空间、或者微博之类的奇葩内容审查规则，对于平台风格以及评论都可以自己把握，建设多样化内容板块，展示自己完整的生活状态。

　　在此之前，我经常会犹豫，心里有些事情不知道怎么分享出来，空间、朋友圈，有些事情发出来总令人尴尬，感觉很别扭，而且过了一段时间，就不会有人关注。我不喜欢发日常流水账，分享的很多都是有特殊意义的。所以博客可能更适合我吧，也可以学习建站运维的知识。

　　生活中，我喜欢电影、摄影、美剧、数码搞机、深度评论以及有趣的思想或文章，多样化的兴趣意味着可以有丰富的拓展空间，运营博客做相关的内容也可以促进这些兴趣的深度发展，既是帮助自身积累，也是对外的展示窗口。如果能给访客带来知识、乐趣或者启发感悟，不失为一种贡献和价值。
