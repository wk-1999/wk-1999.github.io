Part 1 在新设备接管旧站点

一、工具安装

1、安装 node.js：

node 下载地址：http://nodejs.cn/download/

2、安装版本控制工具 Git :

Git 下载地址：https://git-scm.com/download

两个工具安装全部是下一步！下一步！下一步！

3、安装Hexo

npm install hexo-cli -g

通过win+r输入cmd，再敲入安装命令

4、检查安装是否成功

均通过Git Bash检查

二、通过Git Bash生成ssh

1、生成SSH公钥

2、进入 C:\Users\主机名.ssh 目录

复制 id_rsa.pub 里面的信息到Github的 SSH公钥 中

3、Git Bash绑定（直接键入命令）

4、测试ssh是否绑定成功

ssh -T git@github.com

如果一切无误，到此即可在另一台电脑上同样的博客数据文件夹（通过坚果云同步）中进行操作。

Part 2 （如果）建立新站点

一、Github

1、建立账户

2、创建仓库（仓库名是用户名）

仓库分支（branch）目前默认main，后续操作有可能会上传到master，建议如果发生这种情况，就删除main分支，将master设为唯一默认分支。

1、项目文件生成
